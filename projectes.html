🏥 Projecte Hospital

Sistema complet de gestió hospitalària desenvolupat amb Python i PostgreSQL. Inclou gestió de pacients, metges, infermers, visites, operacions, quiròfans, habitacions, medicaments, generació d'informes i exportació de dades en XML. Amb autenticació xifrada mitjançant Fernet i sistema de rols amb permisos.

Descarregar Codi Veure Documentació

Gestió de personal

Alta de metges, infermers i personal vari amb assignació de rols PostgreSQL i permisos específics.

Pacients i visites

Registre de pacients, historial de visites, diagnòstics, medicaments receptats i ingressos hospitalaris.

Operacions i quiròfans

Planificació d'operacions amb reserva de quiròfans, equipament mèdic i assignació de personal.

Seguretat i xifrat

Autenticació segura amb credencials xifrades amb Fernet. Creació de rols i permisos a PostgreSQL.

Informes i rànkings

Rànking de metges per visites, rànking de malalties, informes de personal i plantes hospitalàries.

Exportació XML

Generació de fitxers XML amb dades de visites en un rang de dates, usant funcions XML de PostgreSQL.

Stack Tecnològic

Python 3.11 PostgreSQL psycopg2 Fernet (cryptography) lxml / etree CSV Faker (Dummy Data) Alta disponibilitat
